SIAM Enterprise Model
One example of a SIAM model that has been broken down into components. This is based on the UK Public Sector’s SIAM Enterprise Model, which was first created in 2012. This model and variations of it have been widely adopted in UK public sector organizations. The example model can be used when designing or reviewing specific SIAM models to initiate detailed review and discussion, and to ensure that all of the aspects included are considered. Other SIAM models are in use, but many of them share a similar set of components. Integrating services is in itself a service, and each component of this SIAM model can be delivered as a service. The components must be fully integrated. For example, although IT information security includes security incident and event management, this must be fully interfaced to the incident management process of the Service Desk, and the event management process of the Operations Bridge so that security incidents and events can be managed through the lifecycle, irrespective of how they are initially detected and reported.
